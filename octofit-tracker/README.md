# Octofit Tracker

This workspace contains the Octofit Tracker multi-tier application.

## Frontend

- React 19 with Vite
- Bootstrap for styling

## Backend

- Node.js + Express + TypeScript
- MongoDB via Mongoose

## Development

### Frontend

cd octofit-tracker/frontend
npm install
npm run dev

### Backend

cd octofit-tracker/backend
npm install
cp .env.example .env
npm run dev

### Ports

- Frontend: 5173
- Backend: 8000
- MongoDB: 27017
